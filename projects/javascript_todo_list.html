JavaScript ToDo List

This is a simple to-do list application that allows users to add, delete and reorder items on the list.

Getting Started

To use the application, simply open the index.html file in your browser. You can then start adding items to your to-do list.

Usage

Adding an Item

To add a new item, type the item into the input field and click the + button. The new item will be added to the bottom of the list.

Marking an Item Off

To mark an item off the list, double click the item. The item will be crossed out to indicate that it has been completed.

Deleting an Item

To delete an item, click on the X button next to the item. The item will be removed from the list.

Reordering Items

To reorder items on the list, simply drag and drop them to the desired position.
