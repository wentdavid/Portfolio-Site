React Calendar Meet App

Meet App is a web application that displays upcoming events for developers around the world, built using React. The application fetches event data from the Google Calendar API and displays the information in a user-friendly way. The app demonstrates the use of components, asynchronous programming, APIs, and OAuth2 authentication. You can try the application [here] (insert the link to the live app).

Purpose & Context

The purpose of the Meet App is to create a responsive project that showcases expertise in React, Google Calendar API, and OAuth2 authentication while offering a user-friendly platform for developers to explore and manage upcoming events worldwide.

Objective

The primary objective of the project is to build a complete, responsive, and user-friendly application with React that allows users to explore and manage upcoming events for developers while demonstrating skills in front-end development, API integration, and authentication.

Approach

  1. Project Dependencies: The project utilized React, React Router, Google APIs, Recharts, AWS Lambda, and other libraries for development.
  2. Code Organization: The code was organized into components, providing a clear structure and separation of concerns.
  3. API Integration: A custom API was developed to interact with the Google Calendar API and handle event data retrieval and authentication.
  4. Authentication: Google OAuth2 was implemented to allow users to access their Google Calendar.

Challenges

The primary challenge in building the Meet App was integrating the Google Calendar API, managing the authentication flow, and creating a seamless user experience. By dedicating time to understanding the API documentation and implementing best practices, these challenges were overcome, resulting in a highly functional application.

Duration

The development process took longer than initially anticipated due to the learning curve associated with the Google Calendar API integration and OAuth2 authentication. However, the extra time invested allowed for a solid understanding of these technologies, resulting in a polished final product.

Project Highlights

The following API endpoints are used by this service:

  • Modular code organization, separating concerns into components for better maintainability and scalability.
  • Responsive design using CSS, ensuring a seamless user experience across devices.
  • Robust API integration, utilizing the Google Calendar API for event data retrieval and OAuth2 authentication for secure user access.
  • Implementation of user authentication and authorization for secure access to personal Google Calendar events.

Conclusion

The Meet App project showcases the potential of React as a powerful client-side library for building responsive and user-friendly web applications. The final product demonstrates expertise in front-end development, API integration, and authentication. The successful completion of this project not only serves as an addition to my professional portfolio but also as a testament to the importance of dedication and continuous learning in the field of web development.
